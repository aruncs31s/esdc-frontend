## 🔄 Easy Customization

### Change a Color? (One line)

```css
/* In variables.css */
:root {
  --blue: #your-new-color; /* Done! Entire site updates */
}
```

### Add a Color? (Three lines)

```css
/* In variables.css */
:root {
  --custom: #color-hex;
  --color-custom: var(--custom);
}
```

### Create Variant? (Easy)

```css
.light-theme {
  --base: #ffffff;
  --text: #000000;
  /* ... */
}
```
