## User Interactions

### 1. Image Gallery

```
User clicks thumbnail → Image switches → Smooth transition
```

### 2. Quantity Selection

```
User clicks +/- or enters number → Quantity updates
```

### 3. Apply Coupon

```
User clicks "Apply" → Coupon applied → Price recalculates
```

### 4. Add to Cart

```
User clicks "Add to Cart" → Item added → Success message
```

### 5. Toggle Wishlist

```
User clicks heart → Heart fills/empties → Color changes → Item saved
```

### 6. Navigate Related Products

```
User clicks "View Details" → Navigate to product → Load new product
```

## Responsive Breakpoints

| Device  | Width     | Layout                   |
| ------- | --------- | ------------------------ |
| Desktop | > 768px   | 2-column                 |
| Tablet  | 480-768px | Stacked with adjustments |
| Mobile  | < 480px   | Full-width stacked       |

## Animation States

- **Hover**: Button color change, shadow effect
- **Click**: Immediate feedback
- **Transition**: 0.2-0.3s smooth animations
- **Active**: Color and icon changes (wishlist)

---

**All visual elements are responsive and work seamlessly across all device sizes!**
